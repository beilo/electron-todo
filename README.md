## 开发 & 构建

### window 构建(未测试)

- windows 7 以上
- Node.js 推荐最新版

```
// 下载
git clone https://github.com/beilo/electron-todo.git

// 下载依赖:在项目根目录执行如下命令
npm i

// 打包win:依赖下载完成后执行
npm run package-win
// 打包mac:依赖下载完成后执行
npm run package-mac
// 打包linux:依赖下载完成后执行
npm run package-linux
```
